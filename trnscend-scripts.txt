
Code to Deploy

Here are the parts, along with their recommended placement:

1. In the <head> section of your HTML (ideally after other analytics scripts like Google Analytics, if applicable):

This loads the core Transcend Airgap.js library and sets up the initial transcend object.

<!-- Transcend's Airgap.js script -->
<script data-cfasync="false" onload="window.airgapLoadEvent=event" src="https://transcend-cdn.com/cm/843a2eb4-77c8-4184-9136-71d50fe29864/airgap.js"></script>

<!-- Inline Transcend Initialization and UI Variables -->
<script>
    try {
        if (!self?.transcend?.ready) {
            self.transcend = {
                readyQueue: [],
                ready(callback) {
                    this.readyQueue.push(callback);
                },
                ...self.transcend,
            };
        }

        self.transcend.ready((transcend) => {
            transcend.setUiVariables({
                privacyPolicy: '<a target="_blank" href=https://www.vsp.com/legal/privacy-statement>Privacy Statement</a>', // <<-- UPDATE THIS URL FOR YOUR SITE
                termsOfUse: '<a target="_blank" href=https://www.vsp.com/legal/terms-and-conditions>Terms and Conditions</a>', // <<-- UPDATE THIS URL FOR YOUR SITE
                classActionWaiver: '<a target="_blank" href=https://www.vsp.com/legal/terms-and-conditions?anchor=dispute-resolution>Class Action Waiver</a>', // <<-- UPDATE THIS URL FOR YOUR SITE
                mandatoryArbitration: '<a target="_blank" href=https://www.vsp.com/legal/terms-and-conditions?anchor=dispute-resolution>Mandatory Arbitration</a>' // <<-- UPDATE THIS URL FOR YOUR SITE
            });
        });
    } catch (e) {
        console.log('transcend exception: ', e);
    }
</script>



2. In the <footer> section of your HTML, right before the closing </body> tag:

This div acts as the container where the Transcend consent manager UI (the cookie banner/modal) will be rendered by the airgap.js script.

<!-- Transcend Consent Manager UI Container -->
<div id="transcend-consent-manager" style="position: fixed; z-index: 2147483647;"></div>

Summary of Actionable Steps for New Site Deployment:

Copy the first HTML block (Airgap.js and inline script) and place it in the <head> of your new website.

Verify/Update the src URL in the Airgap.js script. If your new site is for a different Transcend account, change 843a2eb4-77c8-4184-9136-71d50fe29864 to your new account's Tenant ID.

Update all the href URLs within the setUiVariables section of the inline script to point to the correct legal pages on your new website.

Copy the div tag with id="transcend-consent-manager" and place it in the <footer> of your new website, just before </body>.
